# Security Groups & NACLs - Honest Instructions

## Goal
Practice troubleshooting network connectivity issues using SGs and NACLs.

---

## Steps
1. Deploy template
2. Launch test EC2 in this SG
3. Try SSH and HTTP connections
4. If blocked:
   - Check SG inbound/outbound rules
   - Check NACL inbound/outbound rules
5. Fix issues and document
6. Screenshots of failures and resolutions

---

## Deliverables
- Screenshots of blocked and allowed traffic
- Notes in `security-groups-nacl.txt`
